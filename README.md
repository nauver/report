# CoR Annual Communication Report 2025 — Interactive infographic

Open `index.html` in a browser or deploy the folder to GitHub Pages/SharePoint/static hosting.

Accessibility features included:
- semantic sections and headings;
- keyboard-focusable navigation;
- skip link;
- high-contrast toggle;
- respects `prefers-reduced-motion`;
- visible focus indicators;
- no information conveyed by colour alone.

Design language:
- CoR blue/yellow/white palette;
- card-based dashboard;
- interactive channel cards;
- animated counters and charts;
- reduced-motion fallback.
